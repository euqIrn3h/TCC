# Algoritmo Genético para Síntese de Circuitos Lógicos

Aplicação desenvolvida em Python utilizando Kivy para demonstrar a síntese automática de circuitos lógicos através de algoritmos genéticos.

## Funcionalidades

- Geração de circuitos lógicos a partir de mintermos
- Avaliação por tabela verdade
- Otimização do circuito (número de nós ativos)
- Exibição da expressão booleana resultante

## Como executar

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
